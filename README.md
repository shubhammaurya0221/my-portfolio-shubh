# My Portfolio

## Overview
This is a personal portfolio website built using the React.js. It showcases my projects, skills, and experiences, providing a platform to demonstrate my expertise and connect with potential collaborators or employers.

## Tech Stack
- **Frontend**: React, Vite
- **Styling**: CSS, GSAP, Framer-Motion
version

## Features
- Interactive hero section with custom cursor.
- About section detailing personal and professional background.
- Projects section showcasing completed and ongoing projects.
- Technologies section listing skills and tools.
- Contact form for reaching out.
- Smooth animations and responsive design.

## Packages Used
- **React**: For building the user interface.
- **Vite**: For fast development and build tooling.
- **ESLint**: For maintaining code quality.

## Errors Encountered and Solutions
1. **Error**: Issues with Vite's hot module replacement (HMR).
   - **Solution**: Ensured all dependencies were up-to-date and resolved conflicts in the `vite.config.js` file.
2. **Error**: CSS not loading properly in some components.
   - **Solution**: Verified import paths and ensured proper cascading of styles.
3. **Error**: ESLint configuration conflicts.
   - **Solution**: Customized the `eslint.config.js` file to align with project requirements.

## Conclusion
This portfolio website serves as a dynamic and interactive platform to showcase my skills and projects. It was a great learning experience, and I overcame several challenges to deliver a polished product. Future iterations will include backend integration and additional features.
